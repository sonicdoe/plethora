# Plethora

> Jekyll theme for simple blogs

## Installation

1. Add Plethora to your dependencies in your `Gemfile`:

```ruby
gem "plethora", github: "sonicdoe/plethora"
```

2. Set the theme in your `_config.yml`:

```yaml
theme: plethora
```

3. Run `bundle install`
